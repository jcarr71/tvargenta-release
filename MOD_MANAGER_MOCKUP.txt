KEY FEATURES:

✓ Status Indicators
  ● Green dot    = Loaded and running
  ○ Grey dot     = Disabled
  ◐ Spinning dot = Loading/Initializing
  ✗ Red dot      = Error state

✓ Toggle Switches
  [ON]  = Mod enabled (will load on restart)
  [OFF] = Mod disabled (won't load on restart)

✓ Settings Button (⚙️)
  Click to expand settings for that mod
  Configure mod-specific options (rewind speed, intervals, etc.)
  Only available when mod is enabled

✓ Status Messages
  Each mod shows current status and why it might be disabled

✓ Dependency Display
  Shows what each mod requires (ffmpeg, nfcpy, etc.)

✓ Quick Actions
  - Refresh: Reload mod list
  - Restart App Now: Apply changes immediately
  - Install Mod: Browse/search for new mods to install (future)

═══════════════════════════════════════════════════════════════════════════════

INTERACTION FLOW:

1. User sees list of all mods with current status
2. User toggles mod ON/OFF
3. UI updates to show "restart required" state
4. If settings available, user clicks ⚙️ to configure
5. User clicks "Restart App Now" to apply changes
6. App restarts with new mod configuration
