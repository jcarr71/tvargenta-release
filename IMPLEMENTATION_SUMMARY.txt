📝 HOW TO USE

   1. Access Mod Manager:
      → Navigate to: http://[tvargenta-ip]/mod_manager

   2. View Installed Mods:
      → All 3 mods will be listed with current status
      → VCR and Scheduler are enabled by default
      → EQ is disabled by default

   3. Toggle a Mod:
      → Click the toggle switch
      → Switch will update
      → Status banner shows "Restart required"

   4. Apply Changes:
      → Click "Restart App Now" button
      → App will restart with new mod configuration

   5. Monitor Logs:
      → Check logs/tvargenta.log
      → Look for [MOD] entries for detailed logging
