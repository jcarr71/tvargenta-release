## 🚀 How to Use

### Access the Patch Manager
```
http://localhost:5000/patches
```

### Create a Patch
```bash
python3 generate_patch.py old_version/ new_version/ patch.tvpatch \
  --description "Feature update"
```

### Apply a Patch via Web UI
1. Go to `/patches`
2. Drag & drop or browse for `.tvpatch` file
3. Click "Apply Patch"
4. App automatically restarts

### Apply a Patch via API
```bash
curl -X POST -F "patch=@patch.tvpatch" \
  http://localhost:5000/api/patches/apply
```

### Check Version
```bash
curl http://localhost:5000/api/version | jq .
```

### Rollback
```bash
curl -X POST http://localhost:5000/api/patches/rollback
```
